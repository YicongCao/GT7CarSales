# GT7 Car Sales

本工程使用从 [ddm999](https://ddm999.github.io/gt7info) 获取的 GT7 每日车辆数据，包含二手车、传奇赛车，并将其格式化并推送到企业微信 Bot。
  
This project fetches daily GT7 car data (including used and legendary cars) from [ddm999](https://ddm999.github.io/gt7info), formats it, and pushes it to a WeCom (WeChat Work) Bot.

所有代码均使用 GitHub Copilot 生成，全部提示词包含在文件 prompts.txt 中。

All code is generated by GitHub Copilot, and all prompts are included in the file prompts.txt.

`config.json` ：

```json
{
  "wxBotAPIKey": "your-api-key",
  "enableWxBot": true,
  "style": "table"
}
```

- `wxBotAPIKey`：企业微信机器人 API Key
- `enableWxBot`：是否推送到企业微信
- `style`：输出样式，支持 `table`（表格）和 `text`（多行文本）
